== Description ==

Simply allow you to add favicon to your Omega powered site and the WordPress admin via Omega Theme Settings. Developed by [themehall](http://themehall.com "themehall") for [Omega theme](http://themehall.com/omega "Omega theme")

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `/omega-favicon/` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Appearance > Theme Settings", put favicon URL or upload favicon file in Favicon metabox, then click "Update Settings"

Note: You must have [Omega theme](http://wordpress.org/themes/omega "Omega theme") or [child theme](http://wordpress.org/themes/search.php?q=omega "Omega child themes") activated.
